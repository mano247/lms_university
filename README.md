University Management System

A full-stack web application developed as a team project based on university documentation and requirements. The system provides comprehensive management of all university entities, including faculties, students, employees, and administrative staff.

Built with Angular for the frontend and Spring Boot for the backend, with MySQL as the database, the application supports features such as student services, role-based access (admins, staff, students), grading systems, and notification management.

This project demonstrates collaborative development, modular architecture, and real-world academic system design.
